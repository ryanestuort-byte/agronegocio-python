# Sistema de Gestão do Agronegócio

Este projeto tem como objetivo auxiliar no controle de produção agrícola, permitindo o cadastro de culturas, cálculo de área e insumos necessários.

## Funcionalidades:
- Cadastro de culturas agrícolas
- Cálculo automático de insumos
- Armazenamento de dados em JSON
- Listagem de informações

## Tecnologias:
- Python
- Manipulação de arquivos JSON

## Objetivo:
Aplicar conceitos de programação como:
- Funções
- Estruturas de dados
- Manipulação de arquivos

O sistema simula uma solução simples para auxiliar produtores rurais na tomada de decisão.
